## 8) Paleta visual do Lano

Cores dominantes: roxo/lilás (`#5B2BF0`-ish?), fundo claro off-white, cinzas
neutros, acentos em cores suaves por categoria (rosa, azul, verde).

Tipografia: provavelmente **Inter** ou similar (sans-serif clean, geométrica).
Pesos: 400 regular, 600 semi-bold, 800 bold nos titles.

Forma dos cards: **bordas arredondadas grandes** (radius ~12-16px), sombras
sutis, muito whitespace.

Logo: ícone losango/cubo roxo isométrico + wordmark "Lano".

## 10) Gap Analysis — Lano vs VISU-IMOB (estado atual)

### ✅ Já alinhados (feito ou em backlog)

| Feature | VISU-IMOB |
|---|---|
| Multi-tenant com subdomínio | ✅ Fase A |
| CRM > Imóveis (cadastro) | ✅ Wizard 7 passos (feito) |
| CRM > Contatos/Leads | ⏳ Tabela `leads` existe, falta UI (Fase C) |
| Sites white-label | ⏳ Site público básico existe, falta polish |
| Geração Post/Story/Carrusel | ✅ ListaPro Entrega 2 (em teste) |
| Brand kit por tenant | ✅ `tenants.brand_kit` jsonb |
| Signup self-service + planos | ✅ Fase B |

### ❌ Gap — o que Lano tem e a gente NÃO tem ainda

1. **Agentes de IA (40+ prompts soltos)** — hoje só temos o ListaPro disparado pelo cadastro de imóvel. Falta a biblioteca de agentes pra o corretor gerar conteúdo avulso (post sobre mercado, mensagem pro WhatsApp, e-mail de prospecção, etc). **Backlog NOVO.**

2. **Agendar Posts (calendário editorial)** — view de calendário com posts agendados pra publicação automática. **Backlog NOVO.**

3. **Redecorar Ambientes** — upload foto → IA redesenha em estilo X (6 estilos fixos). Usa Flux/Stable Diffusion ou gpt-image-1 modo edição. **Backlog NOVO, mas útil demais.**

4. **CRM > Negócios** — entidade deal (lead × imóvel com pipeline de negociação). **Backlog NOVO.**

5. **Dashboard Home com métricas** — hoje o `/admin` é minimalista. Falta:
   - Cards de performance (créditos usados, posts gerados, imóveis, etc)
   - Ações rápidas
   - Blog integrado
   - Lembretes / tarefas

6. **Sistema de Créditos** — hoje a gente tem `max_imoveis` por plano mas não tem "créditos" granulares consumidos por geração. Precisa refatorar pra modelo de créditos se quisermos replicar Lano.

7. **Blog integrado** — artigos sobre mercado imobiliário. Isso pode entrar junto com a Academia.

8. **Multi-user por tenant (equipe)** — parcialmente feito (`tenant_members` existe), mas falta UI pra convidar/gerenciar equipe.

9. **Importar imóveis** — botão "Importar imóveis" no CRM. Provavelmente CSV ou planilha Excel.

10. **Grupos de contatos** — tags/pastas pra organizar leads.

### 🔒 O que Lano NÃO faz e a gente também NÃO faz no self-service

- **Publicação automática no Instagram/Facebook** — Lano dá templates e permite baixar/agendar, mas quem posta é o cliente. Motivo: Meta API exige app aprovado + token de longa duração por conta, review complexo.
- **Envio automático de WhatsApp** — só gera templates/scripts prontos. Motivo: WABA oficial demora dias pra aprovar, UazAPI/Evolution tem risco de ban.

**Decisão estratégica (Pablo, 2026-04-23):** a Visualis replica esse mesmo
modelo. Self-service = gerar conteúdo + baixar. Automação de postagem/envio é
**serviço de implantação pós-venda** (upsell premium), porque exige:

- Conectar Meta Business API do cliente (manual, precisa review Meta)
- Configurar número WhatsApp no ChatWoot + n8n (a Visualis já tem estrutura)
- Criar webhooks por cliente no n8n pra disparar posts agendados

Isso vira um **add-on de R$ X/mês + taxa de implantação** que fica disponível
só pra clientes do Pro/Premium que escolherem ativar.

### ⭐ Onde VISU-IMOB PODE SER MELHOR que Lano

| Feature | Diferencial |
|---|---|
| **ListaPro completo** — PDF + Post + Story + Carrusel + **Voiceover** + **Vídeo tour** | Lano não tem wizard ponta-a-ponta com vídeo narrado estilo Santiago Muñoz |
| **Academia** (cursos próprios da Visualis) | Lano tem só blog, a gente pode ter cursos completos |
| **WhatsApp automatizado** (ChatWoot + n8n integrado) | Lano só dá templates de mensagem, a gente dispara de verdade |
| **Preço em R$** competitivo (R$ 197 Básico) | Lano preço internacional (USD), a gente agressivo BR |
| **Integração total com CRM da agência** | Visualis já tem fluxo WhatsApp pronto, a gente pluga tudo |
